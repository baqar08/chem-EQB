# chem-EQB
<div align="center">

**chem-EQB is a utility tool for students to help them in balancing their complex chemical equation, calculating molar mass & percentage composition**

</div>

---

## About

chem-EQB is a tool which helps the user in balancing a valid chemical equation, calculate of Molar Mass of complex compounds & finding mass composition of elements and compounds

Visit @ [Link](https://cheemeqb.pythonanywhere.com/)

#link will be live soon

---

## Features

#### Chemical Equation Balancer 
- it can automatically balanced any equation using a matrix based algorithm
#### Molar mass calculater 
- computes accurate molar mass with full periodic table support
#### Percentage composition calculator
- Provides mass percentage of each element in a compound. 
#### Clean & Consistant design
- Modern CSS theme with a periodic-table background across all pages.

## Installation Steps

### Prerequisites
- **Python 3.13+** 
- **pip** 
- **flask** 

---


### Installation Steps

1. **Clone the repo:**
   ```bash
   git clone https://github.com/baqarmustafa84-bot/chem-EQB
   cd chem-EQB
   ```

2. **Create a virtual environment (recommended):**
   
   Makes a seprate environment for the repo
   
   **Windows:**
   ```bash
   python -m venv venv
   .\venv\Scripts\activate
   ```
   
   **macOS / Linux:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

3. **Install dependencies:**
   ```bash
   pip install flask
   ```
   


4. **Run the app:**
   ```bash
   python app.py
   ```

5. **Open it up:**
   
   Go to `http://127.0.0.1:5000` in your browser

## Usage

	
  -  The home page provides quick access to the main chemical equation balancing tool.

 -	Users can navigate between tools using the navigation bar at the top of every page.


 -	The Balance page allows users to enter any chemical reaction and get the balanced equation instantly.
 - The Molar Mass and Percentage Composition pages let users calculate compound masses and element-wise percentages.
 -	Each tool displays results instantly, with clear formatting and error messages whenever an invalid formula is entered.
